# TShirt-Customizer
## Introduction

Create your own style with our new 3D Tshirt Customization App. Pick colors, and add logos to make your virtual Tshirt unique. Built using React.js, Three.js, and TailwindCSS

## Technologies Used

- React.js
- Three.js
- React Three Fiber
- React Three Drei
- Vite
- Tailwind CSS
- Node.js
- Framer Motion
- Valtio

  ## Steps to run

  - Clone the repo
  - Run : npm install
  - RUN : cd client
  - RUN : npm run dev
  - Go to : http://localhost:5173
